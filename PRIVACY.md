# Privacy Policy

**transcript-saver** does not collect, transmit, or store any data. Period.

- No telemetry or analytics
- No network requests
- No data leaves your machine

All transcripts are written locally to your filesystem (`~/.claude/transcripts/` by default). The plugin reads only the session JSONL file provided by Claude Code and writes a Markdown file to disk. Nothing else.
